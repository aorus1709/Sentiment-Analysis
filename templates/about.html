Introduction

Our aim is to identify Hate content of Social Media it can be in Text Image or Video Format First we are going to convert the different datasets like Audio Video to text using different models like Moviepy Speech Recognition Wav2vec
After this we are planning to implement Pre Processing and Cleaning of Text using Vader textBlob Affin etc
Depending upon the Polarity output from Vader Hate content Alert Warning will Popup Or In case Serious issues are detected the message will be transmitted to the nearest Police Station

The Steps we are taking in order to identify the hate content of Social Media is as given below
For converting various types of datasets Audio Video to speech we are planning to implement following steps

  • Step 1 Extract tweets using Tweepy python library
  • Step 2 For Video processing we are planning to convert video to mp3 using Moviepy module and then convert mp3 to text using Speech Recognition Module sttx3
  • Step 3 For Multi Lingual we are planning to use Wav2vec model for speech to text conversion

    • Once the dataset has been converted into text format we are going to use Vader TextBlob Affin SentiwordNet etc modules for Data Cleaning Data Pre processing and to get the Polarity Score
  • Step 4 For Image hatred recognition we are planning to implement Mask RCNN Model which will segments object from image Then we are using Random forest classifier to determine whether the image provoke Hatred Content
  • Step 5 Based on Polarity Score Pop Alert Warning of Hate Content will be displayed
